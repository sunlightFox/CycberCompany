## 本项目采用方式

| 能力 | 采用方式 |
|---|---|
| Agent 编排 | 自研 Chat Runtime + Task Engine，参考 Agents SDK 和 LangGraph |
| 记忆 | 自研分层 Memory，参考 LangGraph、Letta、Mem0、MemGPT、LongMemEval |
| 浏览器 | Playwright MCP 优先，截图和视觉模型作为兜底 |
| 技能 | Skill bundle，包含 `bundle.yaml`、`SKILL.md`、prompts、scripts、tests |
| MCP | 作为外部能力接入协议，不直接暴露全部资产 |
| 安全 | Safety Gate、Approval、Asset Broker、Capability Graph、DLP、Sandbox |
| 回放 | 本地 trace schema + Playwright trace + task artifacts |
