## 过程发现

第一次计划阶段没有执行安装，原因是模型候选解析返回 `finish_reason=length`，候选数为 `0`，系统按安全策略降级为 `manual_only`。

已做通用修复：

- host install 候选解析 prompt 改为要求严格短 JSON、最多 3 个候选。
- 模型候选解析输出上限从 `384` 提到 `1024` token。
- 没有新增飞书专用映射；后续未知软件仍必须通过官方 manifest、可信官网和校验后才能安装。
