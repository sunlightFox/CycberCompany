# 聊天主链路产品测试

测试批次：`CHAT-E2E-20260429`

## 目标

本目录用于沉淀聊天主链路的产品级验收材料。测试覆盖闲聊、复杂聊天、记忆、人格、工具调用、Skill、MCP 和浏览器执行能力，并把每条用例的输入、回复、执行证据和判定结果写入报告。

## 执行环境

- 使用当前仓库的 `data` 环境，不使用临时数据库。
- 测试对象固定为 `mem_xiaoyao` 和当前默认会话。
- 模型场景必须使用真实模型；预检会先验证默认大脑，再跑一条最小聊天确认出现 `model.started` 和 `model.completed`。
- 测试产生的数据统一使用 `CHAT-E2E-20260429` 前缀，便于后续检索。

## 文件说明

- `01-测试用例-闲聊与复杂聊天.md`：闲聊、复杂对话、追问、澄清、上下文延续。
- `02-测试用例-记忆与人格.md`：显式记忆、召回、纠错、敏感记忆、Persona/Heart。
- `03-测试用例-工具Skill-MCP浏览器.md`：任务、工具、审批、Skill、MCP、浏览器执行。
- `run_chat_main_chain_cases.py`：一次性测试执行脚本。
- `04-测试执行报告.md`：脚本生成的执行报告。
- `05-待修复问题.md`：脚本生成的问题清单，只记录问题，不做修复。

## 执行方式

在仓库根目录运行：

```powershell
.\.venv\Scripts\python.exe .\docs\测试\聊天主链路\2026-04-29\run_chat_main_chain_cases.py
```

脚本会在当前 `data` 上执行测试，并覆盖生成 `04-测试执行报告.md` 与 `05-待修复问题.md`。
