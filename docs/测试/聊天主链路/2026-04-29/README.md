# 聊天主链路产品测试

测试批次：`CHAT-E2E-20260429`

## 目标

本目录用于沉淀聊天主链路的产品级验收材料。测试覆盖闲聊、复杂聊天、记忆、人格、工具调用、Skill、MCP 和浏览器执行能力，并把每条用例的输入、回复、执行证据和判定结果写入报告。

## 执行环境

- 使用当前仓库的 `data` 环境，不使用临时数据库。
- 测试对象固定为 `mem_xiaoyao` 和当前默认会话。
- 模型场景必须使用真实模型；预检会先验证默认大脑，再跑一条最小聊天确认出现 `model.started` 和 `model.completed`。
- 测试产生的数据统一使用 `CHAT-E2E-20260429` 前缀，便于后续检索。
