# 飞书附件 100 轮真实模型测试用例

- 入口：飞书渠道 mock connector，经 `poll-once -> channel ingress -> chat turn -> deliver-due`。
- 模型要求：真实大脑模型，逐轮检查 `model.started` 与 `model.completed`。
- 附件类型：PDF、Word、TXT、Excel。
- 附件 oracle：隐藏在真实附件内容中，评分要求回复和产物命中关键事实。
