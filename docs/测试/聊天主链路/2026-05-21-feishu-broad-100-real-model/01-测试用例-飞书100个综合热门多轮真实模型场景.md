# 飞书 100 个综合热门多轮真实模型场景测试用例

- 入口：飞书渠道 mock connector，经 `poll-once -> channel ingress -> chat turn -> deliver-due`。
- 模型要求：真实大脑模型调用，逐轮检查 `model.started` 和 `model.completed`。
- 覆盖：闲聊、记忆、搜索研究、总结、浏览器、办公、系统操作、安全、任务流程、综合多约束。
